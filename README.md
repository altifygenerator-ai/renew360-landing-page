# Renew 360 Cleaning Company LLC landing page

A custom one-page Next.js landing page built from the business information and branding supplied in the Renew 360 flyers and social posts.

## Run locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Build for production

```bash
npm run build
npm start
```

## Main editable files

- `app/page.js` — page content and structured data
- `app/globals.css` — full layout and styling
- `public/images/` — supplied/cropped Renew 360 imagery

## Before launch

Set `NEXT_PUBLIC_SITE_URL` to the final live domain before launch so canonical, social and structured-data URLs use the correct address.
